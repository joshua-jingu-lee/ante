## 1. 설정 페이지 진입

1. 사용자 메뉴에서 "⚙️ 설정" 클릭
2. **Expected**: `/settings` 페이지로 이동
3. **Expected**: 헤더에 "설정" 타이틀 표시

## 2. 시스템 상태 — ACTIVE 상태

1. "시스템 상태" 카드 확인
2. **Expected**: 카드 타이틀 "시스템 상태" 표시
3. **Expected**: 설명 텍스트 "현재 모든 봇의 거래가 정상 운용 중입니다."
4. **Expected**: 우측에 "비상 거래 정지" 버튼 (빨간색 배경) 표시

## 3. 거래소 연결 카드

1. "거래소 연결" 카드 확인
2. **Expected**: 3개 항목 표시
   - 이름: "한국투자증권"
   - 거래 모드: "실전" 뱃지 (파란색/primary)
   - 계좌번호: "50012345-01" (모노스페이스)

## 4. 비상 거래 정지 — 모달 열기

1. "비상 거래 정지" 버튼 클릭
2. **Expected**: 확인 모달 표시
3. **Expected**: 모달 타이틀 "비상 거래 정지" (빨간색)
4. **Expected**: 설명 — "모든 봇의 거래가 즉시 중단됩니다." + "실행 중인 사이클이 완료된 후 정지되며, 보유 포지션은 유지됩니다."
5. **Expected**: 정지 사유 입력 필드 (플레이스홀더: "예: 긴급 시장 변동")
6. **Expected**: 노란색 경고 배너 — "⚠ 이 작업은 실행 중인 모든 봇에 영향을 줍니다."
7. **Expected**: "취소", "거래 정지 확인" 버튼 (거래 정지 확인 버튼 빨간색 배경)

## 5. 비상 거래 정지 — 취소

1. "취소" 버튼 클릭
2. **Expected**: 모달 닫힘

## 6. 비상 거래 정지 — 확인

1. "비상 거래 정지" 버튼 클릭 → 모달 열기
2. 정지 사유: "긴급 시장 변동" 입력
3. "거래 정지 확인" 버튼 클릭

5. **Expected**: 시스템 상태 카드가 HALTED 상태로 변경

## 7. 시스템 상태 — HALTED 상태

1. HALTED 상태의 시스템 상태 카드 확인

4. **Expected**: 우측에 "거래 재개" 버튼 (파란색/positive) 표시

## 8. 거래 재개

3. **Expected**: 각 행에 값 입력 필드 (현재 값 표시)
4. **Expected**: 각 행에 "저장" 버튼 (outline 스타일)
5. **Expected**: 설정 항목 순서 및 값
   - 최대 낙폭 제한 (%): 15.0 — "risk.max_mdd_pct · 이 비율 초과 시 거래 자동 중지 · 예: 15 → 고점 대비 15% 하락 시 정지"
   - 종목당 최대 비중 (%): 30.0 — "risk.max_position_pct · 단일 종목에 투입 가능한 최대 예산 비율 · 예: 30 → 예산의 30%까지 한 종목에 투자"
   - 일일 손실 한도 (%): 5.0 — "rule.daily_loss_limit · 일일 손실이 이 비율 초과 시 전체 거래 정지 · 예: 5 → 당일 손실 5% 초과 시 정지"
   - 총 노출 한도 (%): 20.0 — "rule.max_exposure_percent · 잔고 대비 최대 투자 비율 · 예: 20 → 총 잔고의 20%까지만 투자"
   - 미실현 손실 한도 (%): 10.0 — "rule.max_unrealized_loss · 배정 예산 대비 미실현 손실 초과 시 추가 매수 차단 · 예: 10 → 평가손실 10% 초과 시 매수 불가"
   - 시간당 최대 거래 수: 10 — "rule.max_trades_per_hour · 봇당 시간당 거래 횟수 제한 · 예: 10 → 봇 1개가 1시간에 최대 10회 거래"
   - 거래 허용 시간: 09:00-15:30 — "rule.allowed_hours · 장중 거래 허용 시간대 (KST) · 예: 09:00-15:30 → 오전 9시~오후 3시 30분만 거래"
