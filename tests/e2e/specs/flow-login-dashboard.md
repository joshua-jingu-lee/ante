# Flow: 로그인 → 대시보드

앱의 진입점. 인증 후 대시보드에서 시스템 현황을 확인하는 흐름.

## Seed Data

- Member: `owner` (role: master, status: active)
- Portfolio: 잔고 10,000,000원, 일간 수익 +50,000원 (+0.5%)
- Pending Approvals: 2건 (전략 리포트 1건, 예산 할당 1건)
- System: trading_status = ACTIVE

---

## 1. 비인증 사용자 리다이렉트

1. 로그인하지 않은 상태에서 `/` 접속
2. **Expected**: `/login` 페이지로 리다이렉트
3. "AI-Native Trading Engine" 텍스트와 ANTE 로고 표시

## 2. 로그인 폼 유효성 검증

1. 로그인 페이지에서 아무 입력 없이 "로그인" 버튼 확인
2. **Expected**: 버튼이 비활성 상태 (Member ID, 패스워드 모두 비어 있음)
3. Member ID에 "owner" 입력
4. **Expected**: 버튼이 여전히 비활성 (패스워드 비어 있음)
5. 패스워드에 "wrong" 입력
6. **Expected**: 버튼이 활성화됨

## 3. 패스워드 표시 토글

1. 패스워드 필드에 "testpass" 입력
2. 입력값이 마스킹(***) 표시 확인
3. 패스워드 표시 버튼 (👁) 클릭
4. **Expected**: 입력값 "testpass"가 평문으로 표시됨
5. 다시 클릭
6. **Expected**: 마스킹 복원

## 4. 로그인 실패

1. Member ID: "wrong_user", 패스워드: "wrong_pass" 입력
2. "로그인" 버튼 클릭
3. **Expected**: "ID 또는 패스워드가 올바르지 않습니다." 에러 메시지 표시
4. **Expected**: `/login` 페이지에 머무름

## 5. 로그인 성공 → 대시보드

1. Member ID: "owner", 패스워드: 올바른 패스워드 입력
2. "로그인" 버튼 클릭
3. **Expected**: 버튼 텍스트가 "로그인 중..."으로 변경
4. **Expected**: `/` (대시보드)로 이동
5. **Expected**: 헤더에 "대시보드" 타이틀 표시
6. **Expected**: 헤더 우측에 "🦁 owner · 대표" 사용자 정보 표시
7. **Expected**: 헤더에 시스템 상태 "ACTIVE" + 파란색 인디케이터 표시

## 6. 대시보드 — 포트폴리오 섹션

1. 대시보드 진입 후 포트폴리오 영역 확인
2. **Expected**: 총 자산 "10,000,000원" 표시
3. **Expected**: 일간 수익 "+50,000원 (+0.500%)" 파란색 텍스트로 표시
4. **Expected**: "오늘" 라벨 표시
5. **Expected**: 기간 선택 버튼 5개 (1일, 1주, 1개월, 3개월, 전체)
6. **Expected**: "1주" 버튼이 기본 선택 (파란 배경)

## 7. 대시보드 — 기간 선택 전환

1. "1개월" 버튼 클릭
2. **Expected**: "1개월" 버튼이 파란 배경으로 변경
3. **Expected**: "1주" 버튼이 비활성 스타일로 변경
4. **Expected**: 차트 영역이 갱신됨

## 8. 대시보드 — 승인 대기 섹션

1. 승인 대기 영역 확인
2. **Expected**: "승인 대기" 제목 + "2건" 빨간 뱃지 표시
3. **Expected**: 테이블에 2개 행 표시
4. **Expected**: 각 행에 유형 뱃지(전략 리포트/예산 할당), 제목, 요청일, 승인/거부 버튼 표시
5. **Expected**: "전체 보기 →" 링크가 `/approvals`로 연결

## 9. 대시보드 — 승인 대기 빈 상태

- Seed Data 변경: Pending Approvals 0건

1. 승인 대기 영역 확인
2. **Expected**: "승인 대기 건이 없습니다" 메시지 표시
3. **Expected**: 빨간 뱃지 미표시

## 10. 사이드바 네비게이션

1. 사이드바 확인
2. **Expected**: 7개 메뉴 항목 표시
   - ⭐ 대시보드 (현재 활성 — 파란 배경)
   - 📋 결재함 (뱃지: 2)
   - 💰 자금관리
   - 📈 전략과 성과
   - 🤖 봇 관리
   - 📁 백테스트 데이터
   - 🧑‍💼 에이전트 관리
3. **Expected**: 하단에 "Ante v0.1.0 · MIT" 표시

## 11. 사용자 메뉴

1. 헤더 우측 "🦁" 사용자 아이콘 클릭
2. **Expected**: 드롭다운 메뉴 표시
3. **Expected**: 사용자 이름, 소속 표시
4. **Expected**: "설정" 링크와 "로그아웃" 버튼 표시

## 12. 로그아웃

1. 사용자 메뉴에서 "로그아웃" 클릭
2. **Expected**: `/login` 페이지로 이동
3. **Expected**: 이후 `/` 접속 시 다시 `/login`으로 리다이렉트
