## 1. 백테스트 데이터 페이지 진입

1. 사이드바에서 "📁 백테스트 데이터" 클릭
2. **Expected**: `/backtest-data` 페이지로 이동
3. **Expected**: 헤더에 "백테스트 데이터" 타이틀 표시

## 2. 안내 배너

1. 페이지 상단 안내 영역 확인
2. **Expected**: 파란색 배경 안내 배너 표시
3. **Expected**: "💡 데이터 수집은 Agent에게 요청하거나 CLI(`ante data collect`)를 사용하세요." 텍스트 표시
4. **Expected**: `ante data collect` 부분이 코드 스타일(모노스페이스 배경) 표시

## 3. 데이터셋 테이블 — 헤더

1. "데이터셋" 카드 확인
2. **Expected**: 카드 타이틀 "데이터셋" 표시
3. **Expected**: 우측에 종목 검색 입력 + 타임프레임 셀렉트 표시

## 4. 데이터셋 테이블 — 종목 검색

1. 종목 검색 입력 필드 확인
2. **Expected**: 플레이스홀더 "종목 검색" 표시
3. **Expected**: 입력 시 자동완성 목록 표시 (9개 종목)
4. **Expected**: "삼성전자 (005930)" 등 종목명 (종목코드) 형식

## 5. 데이터셋 테이블 — 타임프레임 필터

1. 타임프레임 셀렉트 확인
2. **Expected**: 4개 옵션 — 전체 타임프레임, 1일, 1시간, 1분
3. **Expected**: "전체 타임프레임"이 기본 선택 상태
4. "1일" 선택
5. **Expected**: 1일봉 데이터셋만 필터링 표시

## 6. 데이터셋 테이블 — 컬럼 및 내용

1. 테이블 컬럼 확인
2. **Expected**: 6개 컬럼 — 종목, 타임프레임, 시작일, 종료일, 행 수, (액션)
3. **Expected**: 행 수 컬럼 우측 정렬
4. **Expected**: 첫 번째 행 — 삼성전자 (005930), 1일, 2020-01-02, 2026-03-12, 1,532, 삭제 버튼
5. **Expected**: 두 번째 행 — 삼성전자 (005930), 1시간, 2024-01-01, 2026-03-12, 12,480, 삭제 버튼
6. **Expected**: 세 번째 행 — 삼성전자 (005930), 1분, 2025-01-01, 2026-03-12, 182,400, 삭제 버튼
7. **Expected**: 각 행에 빨간색 "삭제" 버튼 (outline 스타일) 표시

## 7. 데이터셋 테이블 — 페이지네이션 및 용량 정보

1. 테이블 하단 영역 확인
2. **Expected**: 좌측에 "총 32건 중 1-15 · 전체 1.2 GB (1일봉 340 MB · 1시간봉 580 MB · 1분봉 280 MB)" 표시
3. **Expected**: 우측에 "← 이전", "다음 →" 페이지네이션 버튼 표시
4. **Expected**: 첫 페이지이므로 "← 이전" 버튼 비활성(disabled) 상태
5. **Expected**: "다음 →" 버튼 활성 상태

## 8. 페이지네이션 — 다음 페이지

2. **Expected**: 다음 페이지 데이터셋 표시 (16~32건)

4. **Expected**: 마지막 페이지이면 "다음 →" 버튼 비활성
