# 대시보드

Ante 대시보드는 웹 브라우저에서 시스템 전체를 모니터링하고 관리하는 통합 인터페이스입니다.
자산 현황 확인, 결재 처리, 전략 관리, 봇 제어 등 운용에 필요한 모든 작업을 수행할 수 있습니다.

## 접속 방법

시스템이 실행 중이면 브라우저에서 바로 접속할 수 있습니다.

```
http://<홈서버 주소>:3982
```

로컬에서 실행 중이라면 `http://localhost:3982`으로 접속합니다.

### 로그인

접속하면 로그인 화면이 나타납니다. `ante init`에서 생성한 Master 계정의 Member ID와 패스워드를 입력합니다.

- 인증은 세션 쿠키 기반이며, 24시간 동안 유지됩니다.
- 세션이 만료되면 자동으로 로그인 화면으로 이동합니다.

## 주요 메뉴

사이드바에서 각 메뉴에 접근할 수 있습니다.

### 대시보드

로그인 후 처음 보이는 메인 화면입니다.

- **자산 현황** — 총 자산 가치와 오늘의 손익을 한눈에 확인합니다. 자산 추이 차트로 1일/1주/1개월/3개월/전체 기간의 변화를 볼 수 있습니다.
- **승인 대기** — 에이전트가 제출한 전략 리포트, 예산 할당 요청 등 결재가 필요한 항목이 표시됩니다. 이 화면에서 바로 승인하거나 거부할 수 있습니다.

### 결재함

에이전트가 제출한 모든 결재 항목의 전체 이력을 관리합니다.

- 상태(대기/승인/거부)와 유형별로 필터링할 수 있습니다.
- 결재 유형:
  - **전략 리포트 검토** — 에이전트가 제출한 전략의 채택 여부를 결정합니다.
  - **예산 할당 요청** — 봇에 자금을 배정합니다.
  - **실전 전환 승인** — 모의투자에서 실전투자로 전환합니다.
  - **위험 알림 확인** — MDD 초과, 킬스위치 발동 등 확인이 필요한 건을 처리합니다.

### 자금 관리

계좌별 잔고와 봇별 예산 배분을 관리합니다.

- 상단의 계좌 선택 드롭다운으로 조회할 계좌를 전환합니다.
- 선택한 계좌의 총 잔고, 할당/미할당/예약 금액을 확인합니다.
- 봇별 예산 비중을 차트로 확인하고, 자금을 할당하거나 회수할 수 있습니다.
- 자금 변동 이력(할당, 회수, 체결 등)을 조회할 수 있습니다.

### 전략과 성과

등록된 투자 전략의 목록과 성과를 확인합니다.

- 전략별 상태(운용중/대기/중지), 누적 수익률, 실행 중인 봇을 확인합니다.
- 전략 상세에서는 에쿼티 커브, 승률, 수익 팩터, MDD 등 성과 지표를 분석할 수 있습니다.
- 전략에 귀속된 거래 내역도 조회할 수 있습니다.

### 봇 관리

전략을 실행하는 봇을 생성하고 제어합니다.

- 봇 생성 시 연결할 계좌, 전략, 실행 간격, 예산, 대상 종목을 설정합니다. 봇의 거래 모드는 연결된 계좌의 TradingMode(VIRTUAL/LIVE)에 의해 결정됩니다.
- 계좌 선택 드롭다운으로 특정 계좌에 연결된 봇만 필터링할 수 있습니다.
- 각 봇의 시작/중지/삭제를 제어하고 실행 로그를 확인할 수 있습니다.

### 백테스트 데이터

백테스팅에 사용할 과거 시세 데이터를 관리합니다.

- 수집된 데이터셋(종목, 타임프레임, 기간, 행 수)을 조회합니다.
- 데이터 수집은 에이전트에게 요청하거나 CLI(`ante data collect`)를 사용합니다.

### 에이전트 관리

AI 에이전트를 등록하고 권한을 관리합니다.

- 에이전트의 등록, 정지, 재활성화, 폐기를 관리합니다.
- 권한 범위(scope)를 설정하여 에이전트가 접근할 수 있는 기능을 제어합니다.
- 토큰 재발급 시 기존 토큰은 즉시 무효화됩니다.

### 설정

시스템 전역 설정을 관리합니다.

- **시스템 상태** — Kill Switch로 전체 거래를 즉시 중단하거나 재개합니다.
- **계좌 관리** — 등록된 계좌 목록을 확인하고, 계좌별 상태(ACTIVE/SUSPENDED)를 관리합니다. 계좌별 Kill Switch로 특정 계좌의 거래만 중단할 수 있습니다.
- **거래 설정** — 최대 MDD, 최대 포지션 비중, 일일 손실 한도 등 리스크 규칙을 조정합니다.
- **알림** — 텔레그램 알림 레벨과 대시보드 자동 갱신 간격을 설정합니다.
