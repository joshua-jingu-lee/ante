# D-007: 외부 AI Agent 연동 방식 (2026-03-12)

> Ante 설계 결정 기록.
> 인덱스: [README.md](README.md)

**결정**: CLI 인터페이스 제공, 메인 프로세스 연동 불필요

**Agent 역할 범위**:
- 전략 파일 생성 → 정적 검증 → 백테스트 실행 → 리포트 제출 (여기까지)
- 봇 로드/활성화/운영 관리는 사용자의 영역

**CLI 기능** (모두 메인 프로세스 연동 없이 단독 처리):
- `ante strategy submit` — 전략 파일 정적 검증 + 등록
- `ante backtest run/result` — subprocess 실행 + 결과 조회
- `ante data list/export` — Parquet 직접 읽기
- `ante report submit` — DB 기록

**근거**:
- AI Agent(Claude Code 등)는 CLI를 자연스럽게 다룸
- Agent 필요 기능이 모두 파일/DB 직접 접근으로 완결 → REST API 서버 의존 없음
- 스크립팅 가능, 테스트 용이
- 향후 봇 제어 자동화 시 인증 기반 REST API로 확장 가능

**전략 파일 검증**:
- 정적 검증(AST 기반) 필수: 인터페이스 상속, 필수 메서드, 금지 모듈, 문법 오류
- 런타임 검증(샘플 dry-run) 방식은 추후 결정
