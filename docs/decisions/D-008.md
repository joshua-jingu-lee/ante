# D-008: 웹 대시보드 프레임워크 (2026-03-12)

> Ante 설계 결정 기록.
> 인덱스: [README.md](README.md)

**결정**: FastAPI (백엔드) + React (대시보드) + Lightweight Charts (차트)

**구성**:
- 백엔드: FastAPI — REST API 제공 + 정적 파일 서빙
- 대시보드: React — 웹 대시보드, 빌드 후 정적 파일로 배포
- 차트: TradingView Lightweight Charts — 캔들차트, 수익 그래프 등 금융 차트

**근거**:
- 캔들차트/그래프 등 인터랙티브 차트가 필요하여 SPA 프레임워크 필요
- React는 AI Agent 기반 개발에서 코드 생성 품질이 가장 높음
- Lightweight Charts는 금융 특화 오픈소스, 가볍고 React 래퍼 제공
- 빌드 결과물은 정적 파일 → FastAPI가 서빙, 별도 서버 불필요
- 컴포넌트 라이브러리(shadcn/ui 등) 선택지 풍부
