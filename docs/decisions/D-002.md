# D-002: 아키텍처 패턴 선택 (2026-03-12)

> Ante 설계 결정 기록.
> 인덱스: [README.md](README.md)

**결정**: 대안 A - 이벤트 드리븐 모듈러 모놀리스 (Event-Driven Modular Monolith)

**후보**:
- A: 이벤트 드리븐 모듈러 모놀리스
- B: 계층형 파이프라인 + 코루틴 봇 풀
- C: 경량 메시지 브로커 기반 멀티프로세스

**근거**:
- N100 환경에서 리소스 효율 최고 (단일 프로세스, 단일 이벤트 루프)
- 인메모리 이벤트 전파로 IPC 오버헤드 없음
- EventBus를 통한 느슨한 결합으로 모듈 교체 용이
- 향후 컴포넌트 분리(마이크로서비스화)로의 진화 경로 열려 있음

**참고**: 최종 정리본은 [docs/architecture/README.md](../architecture/README.md)
