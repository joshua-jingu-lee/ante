# 개요

CLI 모듈은 **`ante` 커맨드를 통한 시스템 제어 인터페이스**이다.
사용자 직접 사용과 외부 AI Agent의 시스템 상호작용을 모두 지원한다.

**주요 기능**:
- **시스템 제어**: 시스템 시작·중지·상태 조회
- **봇 관리**: 봇 생성·시작·중지·삭제, 성과·포지션 조회
- **전략 관리**: 전략 검증·제출·목록·상세 조회
- **백테스트**: 백테스트 실행·결과 조회
- **리포트**: 리포트 제출·목록·상세·스키마 조회
- **데이터**: 보유 데이터 목록·스키마·주입
- **Agent 친화적 설계** (D-010): `--format json` 지원, 자기 설명 `--help`, 구조화된 출력
