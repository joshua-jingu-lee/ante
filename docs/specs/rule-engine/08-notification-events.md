# 알림 이벤트 정의 (Notification Events)

Rule Engine이 발행하는 `NotificationEvent` 목록 (category: `"rule"`).

### 일일 손실 한도 초과

**트리거**: `DailyLossLimitRule`에서 일일 손실률이 한도를 초과하여 매수 주문이 거부될 때
**데이터 수집**: 계좌 ID, 봇 ID, 종목, 현재 손실률, 한도, 당일 PnL

```
⚠️ 일일 손실 한도 초과
계좌 domestic의 일일 손실률이 한도를 초과했습니다.
손실률: 5.23% (한도: 5.00%)
당일 손익: -523,000원
추가 매수가 차단됩니다. 매도(손절)는 허용됩니다.
보유 포지션을 확인하고 필요한 조치를 취해주세요.
```

### 총 노출 한도 초과

**트리거**: `TotalExposureLimitRule`에서 총 노출률이 한도를 초과하여 매수 주문이 거부될 때
**데이터 수집**: 계좌 ID, 봇 ID, 종목, 현재 노출액, 한도, 총 자산

```
⚠️ 총 노출 한도 초과
계좌 domestic의 총 포지션 노출이 한도를 초과했습니다.
노출: 22,000,000원 / 한도: 20,000,000원 (총 자산의 20%)
추가 매수가 차단됩니다. 매도(포지션 축소)는 허용됩니다.
포지션 배분을 확인하고 필요한 조치를 취해주세요.
```
