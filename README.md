# Ante — AI 에이전트 기반 자동 매매 시스템

Ante는 개인의 홈서버 위에서 구동되는 자동 매매 시스템입니다.

AI 에이전트가 시장을 조사하고 전략을 고안하면, Ante가 그 전략을 검증하고 봇을 통해 실제 매매를 수행합니다. 거래 결과는 다시 피드백으로 돌아와 다음 전략 개발의 토대가 됩니다. 사용자는 대표로서 전략의 채택과 운용 여부를 최종적으로 결정합니다.

---

## 구조

Ante는 세 역할의 협업으로 작동합니다.

| 역할 | 주체 | 담당 |
|------|------|------|
| 대표 | 사용자 | 전략 채택, 봇 운용, 최종 판단 |
| 직원 | AI Agent | 시장 조사, 전략 고안, 리포트 제출 |
| 인프라 | Ante 시스템 | 전략 검증, 매매 실행, 안전 관리, 성과 피드백 |

---

## Ante가 하는 일

1. **전략 검증** — Agent가 제출한 전략을 정적 분석과 백테스트로 검증합니다.
2. **매매 실행** — 채택된 전략에 따라 봇이 실제 주문을 수행합니다.
3. **안전 관리** — 전역·전략별 거래 규칙으로 손실을 통제하고, 이상 상황 시 자동으로 개입합니다.
4. **성과 피드백** — 거래 기록과 성과 지표를 축적하여 전략 개선의 근거를 제공합니다.

---

## 설계 원칙

- **Agent 잠재성 우선** — 시스템 구조가 AI Agent의 활동을 제약하지 않도록 설계합니다. Ante는 거래 실행과 안전 규칙만 담당하는 얇은 인프라에 머물고, 나머지 판단 영역은 Agent에게 열어둡니다.
- **전략 전권 위임** — 전략의 설계와 수행은 AI Agent에게 최대한 맡깁니다. 전략은 독립 모듈로 존재하며 시스템 코어와 완전히 분리됩니다.
- **인간-AI 공용 인터페이스** — 웹 대시보드, REST API, CLI 등 모든 인터페이스는 사람과 AI Agent가 함께 사용하는 것을 전제로 설계합니다.

---

## 기술 스택

- **언어**: Python (asyncio 기반 단일 프로세스)
- **저장소**: SQLite (운영 데이터) + Parquet (시세 데이터)
- **웹**: FastAPI + React
- **증권사 연동**: 한국투자증권 KIS Open API (초기)
- **운영 환경**: 인텔 N100 홈서버 (4코어, TDP 6W)
